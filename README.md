# Brett Lechtenberg Personal Website 2026

A world-class personal website for **Brett Lechtenberg** — Peak Performance Coach, Speaker, and Author of *The Master's Edge*.

## 🌐 Live Site

**Production:** [View Live Site](https://bl-2026-personal-site-n0tpyx0dh-bretts-projects-3e254e58.vercel.app)

## ✨ Features

- **Bold, Colorful Design** — Dark gradient heroes with animated floating orbs
- **9 Complete Pages** — Homepage, Master's Edge, Speaking, Coaching, AI Advisory, About, Books, Testimonials, Contact
- **Framer Motion Animations** — Smooth scroll animations and hover effects
- **Responsive Design** — Mobile-first, works on all devices
- **Fast Performance** — Next.js 16 with optimized builds

## 🛠 Tech Stack

- **Framework:** Next.js 16.2.2 with App Router
- **UI:** React 19.2.4
- **Language:** TypeScript
- **Styling:** Tailwind CSS 4
- **Animations:** Framer Motion 12
- **Icons:** Lucide React
- **Deployment:** Vercel

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/BrettLechtenbrerg/BL-2026-Personal-Site.git

# Navigate to project
cd BL-2026-Personal-Site

# Install dependencies
npm install

# Start development server
npm run dev

# Open http://localhost:3000
```
